계산 메모장 사용법

기능 소개

계산 메모장은 텍스트를 입력하면 자동으로 숫자와 연산자를 인식하여 계산 결과를 오른쪽에 표시하는 에디터입니다.

사용 방법

  • • 에디터에 숫자와 연산자를 자유롭게 입력하세요.
  • • 텍스트와 숫자가 섞여 있어도 자동으로 인식합니다.
  • • 천 단위 쉼표(,)가 포함된 숫자도 인식합니다.
  • • 계산 결과는 오른쪽에 실시간으로 표시됩니다.

입력 예시

순수 수식:
1000 + 2000 - 500
천 단위 쉼표 포함:
600,000 + 100,000
텍스트와 숫자 혼합:
사과 1000 + 바나나 2000 - 할인 500
여러 숫자 나열:
1000 2000 3000

지원 연산자

+ - * /

계산 결과는 실시간으로 업데이트되며, 입력한 내용을 자동으로 분석하여 최종 합계를 표시합니다.
